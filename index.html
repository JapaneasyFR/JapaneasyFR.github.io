What is Japaneasy?

ようこそ。Welcome to my educational website! Here, you will find different tools to learn Japanese vocabulary with fun. The current version fits very well with beginners, but it aims to be more and more complete. Please give me time :)
If it is you first step on your way to learn Japanese, you should start by learning Kana. You have the list of both hiragana and katakana, and you can then test your knowledge by doing quizzes.

About me
